# SME ERP Backend API

중소기업용 ERP 백엔드 API 프로젝트입니다.  
PHP 실무 경험을 바탕으로 Java Spring Boot 환경에서  
기업용 ERP 시스템 구조와 데이터 흐름을 학습·구현하는 것을 목표로 합니다.

---

## 🎯 프로젝트 목표

- Java + Spring Boot 백엔드 구조 이해
- ERP 핵심 도메인 설계 (회사, 직원, 거래처, 매출)
- 트랜잭션 기반 데이터 정합성 처리
- 실무에 가까운 API 설계 경험

---

## 🛠 기술 스택

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- MySQL
- Gradle
- IntelliJ IDEA

---

## 📦 핵심 도메인 (예정)

- Company (회사)
- User (직원)
- Client (거래처)
- Invoice (매출/매입)
- InvoiceItem (거래 항목)
